# Keyboard Shortcuts

- `Cmd+/`: Command palette

## Navigation
- `Shift+1`: Zoom to fit all
- `Shift+2`: Zoom to fit selection
- `N` / `Shift+N`: Next/Previous Frame
- `Hold Space`: Hand tool

## Create
- `F`: Frame
- `T`: Text Layer

## Layers
- `[` / `]`: Bring to front/back
- `Shift+[` / `Shift+]`: Move one layer to the front/back
- `Cmd+Shift+L`: Lock/Unlock
