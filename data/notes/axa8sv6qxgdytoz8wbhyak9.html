Flow Charts in Mermaid

```mermaid
flowchart TD
    node_a(Node name) --> node_b(Another node)
    node_a --> node_c
```
flowchart TD node_a(Node name) --> node_b(Another node) node_a --> node_c

Orientations

  • TD: Top-Down
  • BT: Bottom-Top
  • LR: Left-Right
  • RL: Right-Left

Styling nodes

```mermaid
flowchart TD
  A:::enemy --> B

  classDef enemy fill:#f00,stroke:#333,stroke-width:4px;
```
flowchart TD A:::enemy --> B classDef enemy fill:#f00,stroke:#333,stroke-width:4px;
