# Create Note Links

## Use the note title as link text
```markdown
[[note-file-without-md]]
```

## Use a custom label as link text
```markdown
[[A Nice Label|note-file-without-md]]
```

## Cross-Vault links

```markdown
[[dendron://vault/note-file-without-md]]
```

Also works with custom labels
