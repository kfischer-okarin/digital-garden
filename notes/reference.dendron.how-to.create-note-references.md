# Create Note References

## Whole note
```markdown
![[name.of.note]]
```

## From one header to the next header

```markdown
![[demo.embed.block#head1:#*]]
```

## Line offset

Skips one line (i.e. the header)

```markdown
![[some.note#header,1]]
```
